# Blog

Personal blog built with [Astro](https://astro.build/).

## Development

```bash
npm install
npm run dev
```

## Write a post

Add a `.md` file to `src/content/thoughts/`:

```markdown
---
title: Post title
date: 2026-04-12
summary: One-line summary shown on the list page.
---

Post body here.
```

## Build

```bash
npm run build    # output in dist/
npm run preview  # preview the build locally
```
